# UI Prototype

This folder contains static HTML/CSS screens for the design stage.

## Run locally

From repository root:

1. cd .pldf/ui-prototype
2. python3 server.py
3. Open http://127.0.0.1:8000/index.html

## Files

- index.html: navigation hub
- login.html: authentication screen
- dashboard-home.html: student home
- learning.html: learning management
- statistics.html: metrics and trends
- activity.html: timeline and reminders
- teacher-dashboard.html: teacher control panel
- styles.css: shared styling
- server.py: local HTTP server

## Notes

- Prototype is static and does not call backend APIs.
- Priority labels are embedded: P1, P2, P3.
