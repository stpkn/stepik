# Product Concept: Programming Course Platform (MVP)

## Problem
Students need a simple way to track progress across programming courses.
Teachers need a compact view of student progress and course performance.

## Target Users
- Student: follows courses, tracks study metrics, checks recent activity.
- Teacher: monitors cohort progress and manages teaching priorities.

## Value
- Unified progress visibility for both roles.
- Reduced friction between learning actions and performance feedback.

## Functional Priorities
- P1: Authentication, student dashboard, learning list, statistics overview, activity feed, teacher dashboard.
- P2: Filtering, trend comparison widgets, actionable recommendations.
- P3: Optional exports, goals, cohort heatmap, reminder tools.

## Platform
- Primary: Web application (desktop + mobile responsive).
